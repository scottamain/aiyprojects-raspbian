aiy.cloudspeech¶

APIs to simplify interaction with the Google Cloud Speech-to-Text service.

Before calling these APIs, be sure you’ve saved your Google Cloud credentials at ~/cloud_speech.json. For more information, see these setup instructions.

This API is designed for the Voice Kit, but it has no dependency on the Voice HAT/Bonnet, so may be used without it.
