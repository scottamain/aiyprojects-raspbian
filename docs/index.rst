AIY Projects API reference
==========================

This is the Python API reference for the AIY Projects library, which is
provided with the Vision Kit and Voice Kit projects.

For more information about the Vision and Voice kits, including
assembly guides and makers guides, go to
`aiyprojects.withgoogle.com <https://aiyprojects.withgoogle.com>`_.
